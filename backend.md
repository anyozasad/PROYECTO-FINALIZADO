# Backend

Node.js + Express

## Carpetas
- src/server.js - Servidor principal
- src/config/ - Configuración BD
- src/controllers/ - Lógica de negocio
- src/routes/ - Rutas API
- src/middlewares/ - Autenticación JWT

## Instalar y ejecutar
```
npm install
npm start
```

Puerto: http://localhost:3000
